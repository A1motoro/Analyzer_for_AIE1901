# STA2002 概率与统计II - Lecture 4: 置信区间I

## 概述
本讲介绍置信区间 (Confidence Interval, CI) 的概念，重点讨论均值的置信区间。

## 置信区间定义

### 基本概念
设 X₁,...,Xn 为样本，X 服从 PDF f(x;θ)，θ ∈ Ω。
设 L = L(X₁,...,Xn), U = U(X₁,...,Xn) 为两个统计量。
给定 α ∈ (0,1)，若
$$1 - α = P_θ (θ ∈ (L, U))$$

则称 (L,U) 为 θ 的 100(1-α)% 置信区间。

**置信系数**: 1-α，表示区间包含真参数的概率。

## 均值置信区间

### 四种情况分析
对总体均值 μ = E[X₁] 的估计：

1. **正态分布，方差已知**: Xᵢ ~ N(μ, σ²), σ² 已知
2. **任意分布，方差已知**: E[X] = μ, Var(X) = σ² 已知
3. **正态分布，方差未知**: Xᵢ ~ N(μ, σ²), σ² 未知
4. **任意分布，方差未知**: E[X] = μ, Var(X) = σ² 未知

### Case 1: 正态分布，方差已知

#### 推导
$$\bar{X} \sim N(\mu, \sigma^2/n)$$
$$P\left(-z_{\alpha/2} \leq \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \leq z_{\alpha/2}\right) = 1-\alpha$$
$$P\left(\bar{X} - z_{\alpha/2}\frac{\sigma}{\sqrt{n}} \leq \mu \leq \bar{X} + z_{\alpha/2}\frac{\sigma}{\sqrt{n}}\right) = 1-\alpha$$

#### 区间形式
$$\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$$

#### 性质
- 以 $$\bar{x}$$ 为中心
- 宽度: 2 z_{\alpha/2} σ/√n
- 样本量 n 越大，区间越窄
- 显著性水平 α 越小，z 值越大，区间越宽

#### 示例
灯泡寿命 X ~ N(μ, 1296)，n=27, $$\bar{x}$$=1478
95% CI: [1464.42, 1491.58]

### Case 2: 任意分布，方差已知

#### 原理
根据中心极限定理 (CLT)，当 n 足够大时：
$$\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \approx N(0,1)$$

#### 近似区间
$$\bar{x} \pm z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}$$

#### 示例
美国人每日橙汁消耗 X，σ=96，n=576，$$\bar{x}$$=133
90% CI: [126.42, 139.58]

### Case 3: 正态分布，方差未知

#### 关键统计量
样本方差: $$S^2 = \frac{1}{n-1} \sum (X_i - \bar{X})^2$$
$$\frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t(n-1)$$

#### 推导
$$P\left(-t_{\alpha/2}(n-1) \leq \frac{\bar{X} - \mu}{S/\sqrt{n}} \leq t_{\alpha/2}(n-1)\right) = 1-\alpha$$
$$P\left(\bar{X} - t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}} \leq \mu \leq \bar{X} + t_{\alpha/2}(n-1)\frac{S}{\sqrt{n}}\right) = 1-\alpha$$

#### 区间形式
$$\bar{x} \pm t_{\alpha/2}(n-1) \cdot \frac{s}{\sqrt{n}}$$

#### 示例
X ~ N(μ, σ²)，n=20，$$\bar{x}$$=507.50，s=89.75
90% CI: [472.80, 542.20]

### Case 4: 任意分布，方差未知

#### 方法1: 当 n 较大时
用 z 代替 t: $$\bar{x} \pm z_{\alpha/2} \cdot \frac{s}{\sqrt{n}}$$

#### 方法2: 适用条件
- n ≥ 100，或
- 各 Xᵢ 近似正态，或
- 样本分布不太偏斜

## 连续性修正 (Continuity Correction)

### 离散分布近似正态
当用连续正态分布近似离散分布时：

#### 规则
- P(X ≤ x) ≈ Φ((x + 0.5 - μ)/σ)
- P(X ≥ x) ≈ 1 - Φ((x - 0.5 - μ)/σ)
- P(a ≤ X ≤ b) ≈ Φ((b + 0.5 - μ)/σ) - Φ((a - 0.5 - μ)/σ)

### 示例
抛硬币 n=1000 次，θ=0.6
求 P(550 ≤ Y ≤ 625)，其中 Y ~ Bin(1000, 0.6)

E[Y]=600, Var(Y)=240
$$P(550 \leq Y \leq 625) = P\left(\frac{549.5-600}{\sqrt{240}} \leq Z \leq \frac{625.5-600}{\sqrt{240}}\right) \approx 0.9496$$

## 单侧置信区间 (One-sided CI)

### 左单侧 (Lower bound)
H₀: μ = μ₀ vs H₁: μ > μ₀
$$P\left(\frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \leq z_\alpha\right) = 1-\alpha$$
$$P\left(\mu \geq \bar{X} - z_\alpha \frac{\sigma}{\sqrt{n}}\right) = 1-\alpha$$
区间: $$(\bar{x} - z_\alpha \frac{\sigma}{\sqrt{n}}, \infty)$$

### 右单侧 (Upper bound)
H₀: μ = μ₀ vs H₁: μ < μ₀
区间: $$(-\infty, \bar{x} + z_\alpha \frac{\sigma}{\sqrt{n}})$$

## 难点讲解

### 1. 置信区间的直观理解
置信区间不是概率区间，而是随机区间。100次重复实验中，约有95次区间包含真参数。

**常见误解**: "95%置信区间有95%概率包含参数"
**正确理解**: "在所有可能的样本中，95%的样本产生的区间包含参数"

### 2. t分布 vs 正态分布
- **方差已知**: 用正态分布 (z)
- **方差未知**: 用t分布，自由度n-1
- **n越大**: t分布越接近正态分布

### 3. 中心极限定理的应用条件
CLT要求独立同分布，但对分布形状要求宽松。实际中n≥30通常足够。

### 4. 连续性修正的必要性
对于离散分布，用±0.5调整可以得到更好的近似。这是因为正态分布是连续的，而离散分布有间隙。

### 5. 单侧 vs 双侧区间
- **双侧**: 对称，适用于"参数等于μ₀吗？"
- **单侧**: 只给边界，适用于"参数大于μ₀吗？"

### 6. 区间宽度的影响因素
- **样本量**: n↑ → 宽度↓ (1/√n)
- **变异性**: σ↑ → 宽度↑
- **置信水平**: 1-α↑ → 宽度↑

## 总结表

| 分布 | 方差 | 置信区间 | 检验统计量 |
|------|------|----------|------------|
| 正态 | 已知 | $$\bar{x} \pm z_{\alpha/2} \sigma/\sqrt{n}$$ | Z ~ N(0,1) |
| 任意 | 已知 | $$\bar{x} \pm z_{\alpha/2} \sigma/\sqrt{n}$$ | Z ≈ N(0,1) |
| 正态 | 未知 | $$\bar{x} \pm t_{\alpha/2}(n-1) s/\sqrt{n}$$ | T ~ t(n-1) |
| 任意 | 未知 | $$\bar{x} \pm t_{\alpha/2}(n-1) s/\sqrt{n}$$ | T ≈ t(n-1) |
