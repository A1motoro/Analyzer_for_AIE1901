## 无偏估计量 (Unbiased Estimator)

### 定义
估计量 u(X₁,...,Xn) 是θ的无偏估计，若
$$E[u(X₁,...,Xn)] = θ$$

否则称为有偏估计 (Biased Estimator)。

### 正态分布示例
设 X₁,...,Xn ~ N(θ₁, θ₂)，参数为(θ₁, θ₂) = (μ, σ²)

**样本均值**: $$\bar{X} = \frac{1}{n} \sum X_i$$
- E[$$\bar{X}$$] = μ = θ₁ ✓ 无偏

**样本方差**: $$S² = \frac{1}{n} \sum (X_i - \bar{X})^2$$
- E[S²] = ? 我们需要计算

### 样本方差的无偏性分析

**已知**:
- $$\bar{X} \sim N(\mu, \sigma^2/n)$$
- $$(n-1)S^2 / \sigma^2 \sim \chi^2(n-1)$$

**因此**:
$$E[S^2] = E\left[\frac{(n-1)S^2}{n-1}\right] = \frac{\sigma^2}{n-1} \cdot E\left[\frac{(n-1)S^2}{\sigma^2}\right] = \frac{\sigma^2}{n-1} \cdot (n-1) = \sigma^2$$

**结论**: S² 是 σ² 的无偏估计，而
$$\hat{\sigma}^2 = \frac{1}{n} \sum (X_i - \bar{X})^2$$
是有偏的，因为 E[$$\hat{\sigma}^2$$] = $$\frac{n-1}{n} \sigma^2$$

## 矩法估计 (Method of Moments)

### 动机
最大似然估计有时难以求出显式解，需要数值优化。矩法提供了一个简单替代方案。

### 矩法原理
对于k维参数 θ = (θ₁,...,θk)：

1. 定义第j阶矩: αⱼ(θ) = E[Xⱼ]  (理论矩)
2. 定义第j阶样本矩: $$\hat{\alpha}_j = \frac{1}{n} \sum X_i^j$$
3. 求解方程组:
   ```
   α₁(θ) = $$\hat{\alpha}_1$$
   α₂(θ) = $$\hat{\alpha}_2$$
   ...
   αk(θ) = $$\hat{\alpha}_k$$
   ```

### 伽马分布示例
X₁,...,Xn ~ Gamma(α, θ)，参数为(α, θ)

**理论矩**:
- α₁ = E[X] = αθ
- α₂ = E[X²] = αθ² + (αθ)² = αθ² + α²θ² = αθ²(1 + α)

**样本矩**:
- $$\hat{\alpha}_1 = \bar{X} = \frac{1}{n} \sum X_i$$
- $$\hat{\alpha}_2 = \frac{1}{n} \sum X_i^2$$

**方程组**:
1. αθ = $$\bar{X}$$
2. αθ²(1 + α) = $$\frac{1}{n} \sum X_i^2$$

**求解**:
从(1)得: θ = $$\bar{X}$$/α
代入(2): α($$\bar{X}$$/α)²(1 + α) = $$\frac{1}{n} \sum X_i^2$$
α ⋅ $$\bar{X}$$²/α² ⋅ (1 + α) = $$\frac{1}{n} \sum X_i^2$$
$$\bar{X}$$²/α ⋅ (1 + α) = $$\frac{1}{n} \sum X_i^2$$

设 V = $$\frac{1}{n} \sum (X_i - \bar{X})^2 = \frac{1}{n} \sum X_i^2 - \bar{X}^2$$

则: $$\bar{X}$$²/α ⋅ (1 + α) = V + \bar{X}^2$$
$$\bar{X}$$² + \bar{X}^2 α = V α + \bar{X}^2 α + \bar{X}^2$$
$$\bar{X}$$² = V α
α = \bar{X}^2 / V
θ = V / \bar{X}

**矩法估计**:
$$\tilde{\alpha} = \frac{\bar{X}^2}{V}, \quad \tilde{\theta} = \frac{V}{\bar{X}}$$

### 泊松分布示例
X₁,...,Xn ~ Pois(λ)

**理论矩**: α₁ = E[X] = λ
**样本矩**: $$\hat{\alpha}_1 = \bar{X}$$

**矩法估计**: $$\tilde{\lambda}_1 = \bar{X}$$

**扩展**: 也可以用二阶矩
E[X²] = λ + λ² = \hat{\alpha}_2
解得: λ² + λ - \hat{\alpha}_2 = 0
λ = [-1 + √(1 + 4\hat{\alpha}_2)]/2

### 均匀分布[0,θ]示例
X₁,...,Xn ~ Uniform[0,θ]

**理论矩**: α₁ = E[X] = θ/2
**样本矩**: $$\hat{\alpha}_1 = \bar{X}$$

**矩法估计**: θ̂ = 2\bar{X}

## MLE vs MoM 比较

| 方面 | MLE | MoM |
|------|-----|-----|
| 理论基础 | 最大化似然 | 匹配矩 |
| 渐进行为 | 通常更有效 | 通常渐近等价 |
| 计算复杂度 | 可能需要数值方法 | 通常有解析解 |
| 唯一性 | 通常唯一 | 可能不唯一 |
| 适用范围 | 广 | 广 |

### 何时选择哪种方法？
- **优先MLE**: 如果有显式解
- **选择MoM**: 计算简单，或MLE无显式解
- **使用最低阶矩**: 优先使用低阶矩构造更稳定的估计

## 难点讲解

### 1. 无偏性的重要性
无偏性保证估计的"公正性"：长期来看，估计值不会系统性地高估或低估真值。

**注意**: 无偏性不是唯一标准！有偏但方差小的估计可能更好 (如$$\hat{\sigma}^2$$ vs S²)。

### 2. 样本方差的无偏性推导
$$\frac{1}{n} \sum (X_i - \bar{X})^2$$ 是有偏的，因为它低估了总体方差。正确的无偏估计需要除以(n-1)。

**直观解释**: 样本方差使用$$\bar{X}$$代替μ，导致低估。损失的1个自由度通过(n-1)修正。

### 3. 矩法的基本思想
矩法将参数估计转化为求解方程：使样本矩等于理论矩。

**关键**: 矩法不需要知道分布的具体形式，只需要知道矩的表达式。

### 4. 伽马分布矩法求解
伽马分布的矩法求解涉及联立方程组。技巧：
1. 用低阶矩表达参数
2. 代入高阶矩方程
3. 注意样本方差 V = $$\frac{1}{n} \sum X_i^2 - \bar{X}^2$$

### 5. 均匀分布的矩法结果
均匀分布[0,θ]的矩法估计是2\bar{X}，而MLE是max{X_i}。

**哪个更好？**
- MLE: 更精确，但可能过拟合
- MoM: 更保守，但可能覆盖不全

### 6. 泊松分布的两种矩法
泊松分布可以用一阶矩(均值)或二阶矩估计λ：
- 一阶矩: λ̂ = \bar{X}
- 二阶矩: λ̂ = [-1 + √(1 + 4 \hat{\alpha}_2)]/2

**哪个更好？** 一阶矩更稳定，因为二阶矩受异常值影响更大。
