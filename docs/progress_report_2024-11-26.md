## 📋 项目概述

本项目是一个现代化的数据分析Web应用，采用React + TypeScript + Vite技术栈构建，提供完整的统计分析、数据可视化和AI辅助分析功能。应用支持中英文双语，采用VSCode Monokai配色方案，提供流畅的用户体验。

**在线访问地址**:
- 正式版本: https://a1motoro.github.io/Analyzer_for_AIE1901/
- 开发预览: https://a1motoro.github.io/Analyzer_for_AIE1901/develop/
