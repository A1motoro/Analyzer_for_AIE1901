**报告日期**: 2025年11月12日

## 📊 项目概述

数据分析师Web应用是一个现代化的数据分析工具，专为统计学习和研究设计。支持多种数据输入方式，提供完整的统计分析功能，并配备直观的可视化界面。

---

## ✨ 核心功能

### 1. 数据输入
- **CSV文件上传**: 支持拖拽上传或点击选择，实时预览数据
- **分布数据生成**: 
  - 正态分布（可自定义均值和标准差）
  - 均匀分布（可自定义最小值和最大值）
  - 指数分布（可自定义参数λ）
  - 泊松分布（可自定义速率）
- **AI数据生成**: 集成AI模型智能生成模拟数据

### 2. 统计分析功能

#### 基本统计
- 均值、中位数、众数
- 方差、标准差
- 偏度、峰度

#### 高级分析
- **参数估计**: 
  - 最大似然估计（MLE）
  - 矩估计（MoM）
  - 支持多种分布类型
  
- **假设检验**:
  - 单样本t检验
  - 两样本t检验
  - 比例检验
  - 支持单侧和双侧检验
  
- **置信区间**:
  - 均值置信区间（z检验和t检验）
  - 比例置信区间（Wilson方法）
  - 可自定义置信水平

- **功效分析** ⭐ **新增功能**
  - 检验功效计算
  - 样本量计算
  - **Power Function 图表** ⭐ **最新更新**
    - 功效 vs 效应量曲线图
    - 功效 vs 样本量曲线图
    - 直观展示功效随参数变化趋势
    - 结合 STA2002 Lecture 9 课程内容

### 3. 数据可视化
- **直方图**: 展示数据分布情况
- **散点图**: 展示数据点分布
- **Power Function 曲线图**: 展示功效函数变化趋势

### 4. AI智能助手
- 集成AI聊天功能
- 提供统计分析建议和数据解读
- 支持中英文对话

### 5. 用户界面
- **深色主题**: 采用 VSCode Monokai 配色方案，护眼舒适
- **响应式设计**: 适配不同屏幕尺寸
- **多语言支持**: 中文/英文切换
- **流畅动画**: 提升用户体验

## 🎯 用户使用场景

### 场景1: 统计分析学习
- 上传课程数据或生成模拟数据
- 查看基本统计量
- 进行参数估计和假设检验
- 理解置信区间的含义

### 场景2: 研究设计
- 使用功效分析计算所需样本量
- 查看 Power Function 图表，了解不同参数对检验功效的影响
- 优化实验设计参数

### 场景3: 数据探索
- 上传自己的数据文件
- 快速查看数据分布
- 进行初步统计分析
- 使用AI助手获得分析建议

## 💡 使用建议

1. **首次使用**:
   - 建议先尝试"分布数据生成"功能，熟悉界面
   - 查看"学习教程"了解功能说明
   - 尝试上传示例数据文件

2. **功效分析**:
   - 使用 Power Function 图表理解功效概念
   - 调整参数观察功效变化
   - 结合课程内容（STA2002 Lecture 9）深入学习

3. **数据输入**:
   - CSV文件应包含数值列
   - 分布生成可自定义参数，建议尝试不同参数值
   - AI生成适合快速获取模拟数据
